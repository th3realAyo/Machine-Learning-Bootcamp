## 🛠️ Setup & Installation

### Prerequisites
- Python 3.8+
- Jupyter Notebook / JupyterLab

### Install dependencies
```bash
pip install -r requirements.txt
```

### Launch Jupyter
```bash
jupyter notebook
```
